# Посібник користувача (для вчителів)

Цей документ пояснює як користутись Telegram-ботом для подачі ранкових звітів відсутності.

## Початок роботи та реєстрація

1. Перейдіть до Telegram-бота. 
2. Натисніть кнопку `Розпочати` (або `Start`) внизу екрана.
3. Бот попросить вас зареєструватись. Виконуючи інструкції від бота по черзі надішліть йому такі дані:
    - ваше ПІБ;
    - назву класу (наприклад, 5-А або 10-Б);
    - загальну кількість учнів у вашому класі за списком.

> Реєстрація виконується лише один раз. Надалі система вас пам'ятатиме.

## Подача щоденного звіту

1. Для подачі звіту натисніть на кнопку `✍️ Створити звіт`.
2. По черзі напишіть у повідомленнях кількість відсутніх та хворих учнів.

> Бот автоматично надішле вам повідомлення о 08:55, якщо ви забудете подати звіт.

## Налаштування профілю

Якщо при реєстрації ви допустили помилку або у вас зміниться кількість дітей у класі, то в будь-який момент ви можете змінити дані.

1. Натисніть кнопку `👤 Профіль`
2. Натисніть `✏️ Змінити` та введіть нові дані.
