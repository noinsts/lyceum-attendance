# 🗄️ Структура бази даних 

Усі таблиці в проєкті наслідуються від базового класу, який автоматично додає до кожного запису такі поля:

| Поле       | Тип даних | Опис                                   |
| ---------- | --------- | ---------------------------------------|
| created_at | DateTime  | Дата і час створення запису            |
| updated_at | DateTime  | Дата і час останнього оновлення запису |

## 👥 Таблиця `users` (Користувачі)

Зберігає інформацію про зареєстрованих користувачів.

| Поле    | Тип даних  | Опис                                        | Зв'язки / Ключі  |
| ------- | ---------- | ------------------------------------------- | ---------------- | 
| user_id | BigInteger | Telegram ID користувача                     | PK (Primary Key) |
| name    | String     | Повне ім'я                                  |                  |
| form    | String     | Клас, класним керівником якого є користувач |                  |

## 🏫 Таблиця `forms` (Класи)

Зберігає інформацію про класи та кількість дітей у них.

| Поле           | Тип даних | Опис                    | Зв'язки / Ключі  |
| -------------- | --------- | ----------------------- | ---------------- | 
| name           | String    | Назва класу             | PK (Primary Key) |
| students_count | Integer   | Кількість учнів в класі |                  |

## 📊 Таблиця `reports` (Звіти)

Зберігає щоденні звіти про відсутність від вчителів.

| Поле       | Тип даних  | Опис                    | Зв'язки / Ключі  |
| ---------- | ---------- | ----------------------- | ---------------- | 
| id         | BigInteger | Унікальний ID репорту   | PK (Primary Key) |
| form       | String     | Назва класу             |                  |
| date       | Date       | Дата репорту            |                  |
| absentees  | Integer    | Кількість відсутніх     |                  |
| patients   | Integer    | Кількість хворих        |                  |
| total      | Integer    | Кількість учнів у класі |                  |

## 👑 Таблиця `admins` (Адміністратори)

Список користувачі, які мають доступ до панелі керування та зведеної статистики.

| Поле    | Тип даних  | Опис                    | Зв'язки / Ключі  |
| ------- | ---------- | ----------------------- | ---------------- | 
| user_id | BigInteger | Telegram ID користувача | PK (Primary Key) |
| name    | String     | Ім'я адміністратора     |                  |
